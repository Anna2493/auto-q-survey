# autoQsurvey

https://auto-q-survey-web.herokuapp.com/

2021 Honours project

How to run this project:
1.	Navigate to auto-q-survey folder 
2.	Run npm instal command to instal the dependencies
3.	Run the node server.js command to start to server
4.	Open up new terminal and navigate to client folder
5.	Run npm install command to install the client dependencies 
6.	Run npm start command
